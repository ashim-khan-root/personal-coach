# Market Research

## Workflow

1. **Define the market** — geography, segment, customer profile
2. **Size the opportunity** — TAM → SAM → SOM estimate
3. **Map competitors** — direct, indirect, future
4. **Identify trends** — growing/shrinking, regulatory, tech shifts
5. **Synthesize** — opportunity score, recommendation

## Sources

- Web search + deep_research tool for broad landscape
- Competitor sites for positioning, pricing, features
- Review sites (G2, Capterra) for pain points
- Reddit, forums for unfiltered customer sentiment
- Google Trends for trajectory

## Output

Structured markdown with: market definition, size estimates, competitor matrix (3-5 key players), trends, recommendation (go/no-go/pivot).
