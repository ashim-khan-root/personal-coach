# STARFOX SECURITY — Company Profile

STARFOX SECURITY is an MOI-SSD approved security systems integrator based in Doha, Qatar. We specialize in the design, supply, installation, and maintenance of IP-based CCTV surveillance systems, ANPR/LPR systems, access control, and smart home security solutions.

## Our MOI Compliance Process
1. **Site Survey** — Free visit to assess coverage and compliance
2. **System Design** — AutoCAD layouts showing camera positions, coverage, angles
3. **MOI Submission** — Handle full SSD portal submission for design approval
4. **Installation** — Certified technicians, MOI-approved equipment only
5. **Testing & Commissioning** — FAT, SAT, system verification
6. **MOI Inspection** — Arrange and pass MOI SSD inspection
7. **Handover + AMC** — Compliance certificate, maintenance contract
