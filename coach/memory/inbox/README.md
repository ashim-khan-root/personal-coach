# Inbox

Capture zone for loose thoughts, tasks, and ideas before they get organized.

## How to use
1. Drop anything here: quick notes, URLs, task ideas, meeting notes
2. Run `python tools/inbox_processor.py` to organize everything into the right place
3. Items get moved to: sessions, goals, habits, work/, or archived

## Structure
- `captures/` — raw unprocessed items (default landing spot)
- `processed/` — items that have been organized (archive)
