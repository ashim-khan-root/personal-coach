# Job Search Guide — Qatar Tech Roles

## NaukriGulf (naukrigulf.com)

1. Go to site, search: `Java Full Stack` or `React Node.js Developer`
2. Filter by **Location: Qatar**
3. Create account → Upload CV → Click "Easy Apply" on matching roles
4. Set up **Job Alert** for "Full Stack Developer Qatar" to get email notifications

## Bayt (bayt.com/en/qatar/)

1. Search: `Software Engineer` or `Full Stack Developer`
2. Filter by **Country: Qatar**
3. Create profile (fill all sections for higher visibility)
4. Use "Easy Apply" buttons
5. Set up email alerts

## Pro Tips

- Apply within 24h of posting for best response rate
- Customize the CV for each role if the JD emphasizes specific skills
- Keep LinkedIn profile updated and set to "Open to Work"
