## Objective

Build a free online tools website that generates daily organic traffic → ad revenue ($50-500+/mo in 6-12 months). Zero hosting cost, Hugo + Cloudflare Pages, $10/yr domain only.

## Architecture

```
tools.example.com (MAIN site — NOT subdomain, example.com/)
├── /tools/
│   ├── image-resize/
│   ├── qr-generator/
│   ├── json-formatter/
│   ├── word-counter/
│   ├── color-picker/
│   ├── base64-encoder/
│   ├── text-diff/
│   ├── regex-tester/
│   ├── password-generator/
│   └── unit-converter/
├── /calculators/
│   ├── loan-calculator/
│   ├── bmi-calculator/
│   └── age-calculator/
├── /other-categories/
└── /blog/ (support content for SEO)
```

**Tech stack:**
- **Static site:** Hugo (modular, fast builds)
- **Hosting:** Cloudflare Pages (free, unlimited bandwidth, 500 builds/mo)
- **Domain:** ~$10/yr (Namecheap/Cloudflare Registrar)
- **Each tool:** Hugo page + single `index.html` with embedded vanilla JS (no frameworks, no npm)
- **Ads:** AdSense → Ezoic (10k sessions) → Mediavine (50k sessions)

**Why Hugo not React/Next:**
- Zero server cost
- No build complexity
- Each tool = standalone HTML page with inline JS
- Cloudflare Pages serves flat HTML instantly
- Easy to add 50+ tools without framework bloat

## Tool Template (each tool = same pattern)

```
content/tools/image-resize/
├── index.md          ← Hugo page with tool metadata
└── tool.js           ← standalone vanilla JS (embedded via shortcode)
```

Every tool page has:
1. RankMath-style SEO meta (title, description, H1, focus keyword)
2. Form/input area
3. Vanilla JS that runs the tool
4. Result output area
5. Ad slots (top, middle, bottom)
6. Related tools section (internal links)

## SEO Strategy

- Every tool page optimized for `tool name + "online" + "free"` keywords
- Interlink all tools (related tools section on each page)
- Create category pages (e.g., `/tools/` lists all tools)
- Blog posts that reference tools (e.g., "How to resize images online free")
- Sitemap auto-generated by Hugo
- Cloudflare Pages serves from edge (fast = SEO bonus)
